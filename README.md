# ペライチ告知機 v1

イベント情報を入力すると、A4縦の告知チラシをブラウザ上で即時生成する静的ツールです。サーバー不要で、GitHub Pages でもそのまま動かせます。

## ファイル構成

- `index.html`
- `style.css`
- `script.js`

## 使い方

1. `index.html` をブラウザで開きます。
2. 左側フォームにイベント情報を入力します。
3. `JSON読込` を押すと、保存済みのJSONファイルを読み込めます。
4. 右側のA4プレビューへ即時反映されます。
5. `JSON保存` を押すと、現在の入力内容をJSONファイルで保存できます。
6. `印刷する` を押すと `window.print()` が開きます。

## 仕様メモ

- PCでは2カラム、スマホでは縦積みです。
- 未入力項目はプレビューで自動非表示です。
- フォームには一部項目の目安文字数を表示しています。
- 問い合わせ先やSNSは、メール・電話・X・Instagram・Webを分けて入力できます。
- 現在の入力内容はJSONファイルとして保存・再読込できます。画像は圧縮した data URL として一緒に書き出されます。
- 画像は1枚だけアップロードできます。
- メイン画像は現在の紙面レイアウトでは `2200 × 850 px` 前後を最大表示サイズの目安としています。
- QRコードURLを入れると、外部CDN読込の `qrious` でQRコードを生成します。
- QRコード下テキストは、QRコードURL入力時のみ有効になります。
- 印刷時はプレビュー部分のみ出力され、A4縦に最適化されます。
- 内容量によってはA4 1ページに収まらない場合があるため、最終確認は必ず印刷プレビューで行ってください。

## GitHub Pages

このフォルダをそのままリポジトリに置き、GitHub Pages の公開対象に設定すれば動作します。ビルド手順は不要です。
